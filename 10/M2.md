#### Eksempel 1

Find grænserne for følgende funktioner:

1. \(\lim\limits_{x \to 0} \frac{2x^3+x^2}{x^2-x}\)
2. \(\lim\limits_{x \to \infty} \frac{3x^2-2x}{x^2+5}\)
3. \(\lim\limits_{x \to 3} \frac{x-3}{x^2-9}\)

??? answer "&nbsp;"

    1. \(\lim\limits_{x \to 0} \frac{2x^3+x^2}{x^2-x} = 0\)
    2. \(\lim\limits_{x \to \infty} \frac{3x^2-2x}{x^2+5} = 3\)
    3. \(\lim\limits_{x \to 3} \frac{x-3}{x^2-9} = \frac{1}{6}\)

#### Eksempel 2: Simple afledte

Find de afledte af følgende funktioner:

1. \(f(x)=x^5\)
2. \(f(t)=t^{-\frac{1}{2}}\)
3. \(y=t^{-2.5}\)

??? answer "&nbsp;"

    1. \(f'(x)=5x^4\)
    2. \(f'(t)=-\frac{1}{2}t^{-\frac{3}{2}}\)
    3. \(y'=-2.5t^{-3.5}\)

#### Eksempel 3: Afledte af potenser
Skriv følgende som potenser og differentier derefter:

1. \(\frac{1}{x^3}\)
2. \(\sqrt{x}\)
3. \(\frac{1}{x \sqrt{x}}\)

??? answer "&nbsp;"

    1. \(\frac{1}{x^3} = x^{-3}\), afledt: \(\frac{d}{dx}x^{-3} = -3x^{-4}\)
    2. \(\sqrt{x} = x^{1/2}\), afledt: \(\frac{d}{dx}x^{1/2} = \frac{1}{2}x^{-1/2}\)
    3. \(\frac{1}{x \sqrt{x}} = x^{-4/3}\), afledt: \(\frac{d}{dx}x^{-4/3} = -\frac{4}{3}x^{-7/3}\)

#### Eksempel 4: Afledte af summer og differencer

Find de afledte af følgende funktioner:

1. \(y=3x^{-5}+\frac{2}{x^3}\)
2. \(f(u)=u^{\frac{4}{3}}-2u^{-5}\)
3. \(g(z)=6z^{-3}-\frac{4}{z}\)

??? answer "&nbsp;"

    1. \(\frac{d}{dx}(3x^{-5}+\frac{2}{x^3}) = -15x^{-6} - 6x^{-4}\)
    2. \(\frac{d}{dx}(u^{\frac{4}{3}}-2u^{-5}) = \frac{4}{3}u^{1/3} + 10u^{-6}\)
    3. \(\frac{d}{dx}(6z^{-3}-\frac{4}{z}) = -18z^{-4} + \frac{4}{z^2}\)

#### Eksempel 5: Afledte af produkter og kvotienter
Brug produktreglen og kvotientreglen til at differentiere funktionerne nedenfor:

1. \(f(x)=(2x^2-1)(x+3)\)
2. \(g(x)=(x^3-2x)(x^2+4)\)
3. \(h(x)=\frac{x+1}{x^2-2}\)
4. \(g(t)=\frac{t^2-1}{t^2+t-2}\)

??? answer "&nbsp;"

    1. \(f'(x) = 6x^2 + 12x - 1\)
    2. \(g'(x) = 5x^4 + 6x^2 - 8\)
    3. \(h'(x)=\frac{-x^2-2x-2}{(x^2-2)^2}\)
    4. \(g'(t)=\frac{t^2-2t+1}{(t^2+t-2)^2} = \frac{(t-1)^2}{(t+2)^2(t-1)^2} = \frac{1}{(t+2)^2}\) for \(t \neq 1\)

#### Eksempel 6: Afledte ved kædereglen
Brug kædereglen til at differentiere funktionerne nedenfor:

1. \((3x-1)^4\)
2. \((x^3-x)^5\)
3. \(f(t)=\sqrt{x^2+4}\)
4. \(z=(x^2 - \frac{1}{x})^3\)
5. \(\frac{x^2}{\sqrt{x^2+1}}\)

??? answer "&nbsp;"

    1. \(12(3x-1)^3\)
    2. \(5(3x^2-1)(x^3-x)^4\)
    3. \(\frac{x}{\sqrt{x^2+4}}\)
    4. \(3(x^2 - \frac{1}{x})^2(2x + \frac{1}{x^2})\)
    5. \(\frac{x^3+2x}{(x^2+1)^{3/2}}\)

#### Eksempel 7: Afledte af eksponential- og logaritmefunktioner

1. \(f(x)=\ln(x^2+1)\)
2. \(f(x)=e^{3x^2}\)
3. \(f(x)=\ln(e^x+1)\)

??? answer "&nbsp;"

    1. \(\frac{2x}{x^2+1}\)
    2. \(6xe^{3x^2}\)
    3. \(\frac{e^x}{e^x+1}\)

#### Eksempel 8: Anvendelser af afledte

En partikels position som funktion af tiden er givet ved $s(t) = 10t^2 - 5t + 20$, hvor $t$ måles i sekunder og $s(t)$ i meter.

1. Find partiklens hastighedsfunktion, $v(t)$.

    ??? answer "&nbsp;"

        Hastighedsfunktion: $v(t) = s'(t) = 20t - 5$ m/s

2. Find partiklens accelerationsfunktion, $a(t)$.

    ??? answer "&nbsp;"

        Accelerationsfunktion: $a(t) = v'(t) = 20$ m/s²

3. Beregn partiklens hastighed og acceleration ved $t = 2$ sekunder.

    ??? answer "&nbsp;"

        Hastighed ved $t = 2$ sekunder: $v(2) = 20(2) - 5 = 35$ m/s

        Acceleration ved $t = 2$ sekunder: $a(2) = 20$ m/s²
